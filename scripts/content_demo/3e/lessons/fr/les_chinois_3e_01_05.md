# Les couleurs, adjectifs et la négation

Savoir décrire le monde qui nous entoure est essentiel en chinois. Dans cette leçon, nous apprenons les couleurs, la place des adjectifs dans la phrase, et les deux particules de négation — l'une des distinctions grammaticales les plus importantes du mandarin.

## Les couleurs (颜色 yánsè)

:::propriete
| Couleur | Caractère | Pinyin | Note culturelle |
|---|---|---|---|
| Rouge | 红色 | hóngsè | Symbole de chance, de bonheur et de prospérité |
| Orange | 橙色 | chéngsè | |
| Jaune | 黄色 | huángsè | Couleur impériale (empereurs) |
| Vert | 绿色 | lǜsè | Nature, espoir, environnement |
| Bleu | 蓝色 | lánsè | Ciel, mer |
| Violet | 紫色 | zǐsè | Noblesse, spiritualité |
| Blanc | 白色 | báisè | ⚠️ Deuil et funérailles en Chine |

| Gris | 灰色 | huīsè | |
| Rose | 粉色 | fěnsè | Féminité |

:::attention
En Chine, le **blanc** est la couleur du deuil (funérailles, maladies). Il ne faut pas offrir des fleurs blanches à un hôte, ni porter du blanc à une fête. C'est l'inverse de la France où le blanc est la couleur de la mariée.

Le **rouge** est la couleur du bonheur — toujours approprié pour les fêtes, les cadeaux et les mariages.
:::

## Les adjectifs et la structure de la phrase

:::definition
En mandarin, les adjectifs peuvent s'utiliser de deux façons :

1. **Comme prédicat** (sans verbe «être») : Sujet + 很 hěn + adjectif
   - 这本书很好。Zhè běn shū hěn hǎo. = Ce livre est bon.
   - 很 hěn (très) est obligatoire devant l'adjectif prédicat pour éviter un sens comparatif.

2. **Comme épithète** (avant un nom) : adjectif + 的 de + nom
   - 红色的包 hóngsè de bāo = le sac rouge
   - 漂亮的女孩 piàoliang de nǚhái = une belle fille
:::

## La négation — 不 bù et 没 méi

:::definition
Le mandarin possède **deux particules de négation** distinctes selon le contexte :

**1. 不 bù** — nie le présent, le futur, les habitudes et les adjectifs
**2. 没 méi** — nie le passé ou le verbe 有 yǒu (avoir/il y a)
:::

:::propriete
| Contexte | Particule | Exemple | Traduction |
|---|---|---|---|
| Action présente / future | 不 bù | 我不去 wǒ bù qù | je ne vais pas (maintenant/plus tard) |
| Habitude / refus | 不 bù | 我不吃辣 wǒ bù chī là | je ne mange pas épicé (habitude) |
| Identité / état | 不 bù | 我不是学生 wǒ bú shì xuéshēng | je ne suis pas étudiant |
| Adjectif | 不 bù | 不好 bù hǎo | pas bon |
| Action passée | 没 méi | 我没去 wǒ méi qù | je ne suis pas allé(e) |
| Possession / existence | 没有 méiyǒu | 我没有钱 wǒ méiyǒu qián | je n'ai pas d'argent |
| Expérience passée | 没 méi | 我没吃过日本菜 | je n'ai jamais mangé japonais |
:::

:::methode
**Comment choisir entre 不 et 没 ?**

1. C'est une action **passée** ? → utiliser **没** méi
   - 我没去学校。Wǒ méi qù xuéxiào. = Je ne suis pas allé à l'école. (hier)

2. C'est une négation de **avoir (有 yǒu)** ? → utiliser **没有** méiyǒu
   - 我没有作业。Wǒ méiyǒu zuòyè. = Je n'ai pas de devoirs.

3. Tout le rest (présent, futur, adjectif, habitude, état) → utiliser **不** bù
   - 我不去。Wǒ bù qù. = Je ne vais pas. (maintenant)
   - 这不好吃。Zhè bù hǎo chī. = Ce n'est pas bon.
:::

:::exemple
Traduis en mandarin :

1. Je ne vais pas à l'école demain.
   → 我明天**不**去学校。Wǒ míngtiān bù qù xuéxiào.

2. Je ne suis pas allé(e) à l'école hier.
   → 我昨天**没**去学校。Wǒ zuótiān méi qù xuéxiào.

3. Je n'ai pas de stylo.
   → 我**没有**钢笔。Wǒ méiyǒu gāngbǐ.

4. Ce sac n'est pas rouge, il est bleu.
   → 这个包**不**是红色的，是蓝色的。
:::

:::retenir
- Couleurs principales : 红(rouge/chance), 黄(jaune/impérial), 绿(vert), 蓝(bleu), 白(blanc/deuil⚠️), 黑(noir).
- Adjectif prédicat : Sujet + **很 hěn** + adjectif (très/assez).
- Adjectif épithète : adjectif + **的 de** + nom.
- **不 bù** = négation présent/futur/habitude/adjectif.
- **没 méi / 没有 méiyǒu** = négation passé / verbe avoir.
- Ne jamais écrire 不有 — toujours **没有** pour «ne pas avoir».
:::
