# Traitement de texte

Le traitement de texte est l'un des usages les plus répandus de l'ordinateur. Maîtriser un logiciel comme LibreOffice Writer ou Microsoft Word est une compétence essentielle pour la vie scolaire et professionnelle.

:::definition
Un **logiciel de traitement de texte** est un programme permettant de saisir, modifier, mettre en forme, enregistrer et imprimer des documents texte. Les plus courants sont **LibreOffice Writer** (gratuit et libre), **Microsoft Word** (payant) et **Google Docs** (en ligne, gratuit).
:::

## L'interface du traitement de texte

:::propriete
L'interface d'un traitement de texte comprend :

- **La barre de titre** : affiche le nom du document et du logiciel.
- **Le ruban / barre d'outils** : regroupe les fonctions de mise en forme (police, taille, styles…).
- **La zone de saisie** : l'espace blanc où l'on tape le texte.
- **La règle** : permet de régler les marges et les tabulations.
- **La barre d'état** : affiche le numéro de page, le nombre de mots, la langue.
:::

## Mise en forme des caractères

:::propriete
La mise en forme des caractères modifie l'aspect du texte sélectionné :

| Mise en forme | Raccourci | Effet |
|---|---|---|
| **Gras** | Ctrl + G (ou Ctrl + B) | **Texte en gras** |
| *Italique* | Ctrl + I | *Texte en italique* |
| Souligné | Ctrl + U | Texte souligné |
| Police | — | Changer la forme des lettres (Arial, Times…) |
| Taille | — | Changer la taille en points (12 pt standard) |
| Couleur | — | Changer la couleur du texte |
:::

## Mise en forme des paragraphes

:::propriete
La mise en forme des paragraphes agit sur les blocs de texte :

| Option | Description |
|---|---|
| Alignement à gauche | Texte aligné sur la marge gauche (défaut) |
| Alignement centré | Texte centré sur la page |
| Alignement à droite | Texte aligné sur la marge droite |
| Justifié | Texte aligné des deux côtés (gauche ET droite) |
| Interligne | Espace entre les lignes (1, 1,5 ou 2) |
| Retrait | Décalage du paragraphe depuis la marge |
:::

## Les styles

:::definition
Un **style** est un ensemble de mises en forme (police, taille, couleur, alignement, interligne…) enregistrées sous un nom et applicable en un seul clic. Les styles les plus courants sont :

- **Normal** : corps du texte standard.
- **Titre 1** : titre principal du document (grand, gras).
- **Titre 2** : sous-titre (un peu plus petit).
- **Titre 3** : sous-sous-titre.

L'utilisation cohérente des styles permet de générer automatiquement une **table des matières**.
:::

:::exemple
Pour structurer un exposé sur l'environnement :
- Sélectionner «Introduction» → appliquer le style **Titre 1**.
- Sélectionner «1. La déforestation» → appliquer le style **Titre 2**.
- Sélectionner le texte de l'exposé → style **Normal**.
- Insérer → Table des matières → le logiciel génère automatiquement la table avec les numéros de page.
:::

## Enregistrement et formats de fichiers

:::propriete
| Format | Extension | Usage |
|---|---|---|
| Document Word | .docx | Format Microsoft Office, très répandu |
| Document ODF | .odt | Format libre (LibreOffice, OpenOffice) |
| PDF | .pdf | Partage final, mise en page préservée |
| Texte brut | .txt | Texte sans mise en forme |

> **Conseil :** Travailler en .odt ou .docx pour modifier le document. Exporter en PDF pour le partager ou l'imprimer.
:::

## Raccourcis clavier essentiels

:::methode
Maîtriser ces raccourcis permet de travailler beaucoup plus vite :

| Action | Raccourci Windows |
|---|---|
| Enregistrer | Ctrl + S |
| Annuler | Ctrl + Z |
| Rétablir | Ctrl + Y |
| Copier | Ctrl + C |
| Couper | Ctrl + X |
| Coller | Ctrl + V |
| Sélectionner tout | Ctrl + A |
| Imprimer | Ctrl + P |
| Gras | Ctrl + G |
| Italique | Ctrl + I |
| Souligner | Ctrl + U |
| Rechercher | Ctrl + F |
:::

:::attention
Penser à enregistrer fréquemment son travail avec **Ctrl + S**. En cas de coupure de courant (très fréquente en contexte camerounais), tout travail non sauvegardé est perdu. LibreOffice dispose d'une récupération automatique, mais elle n'est pas infaillible.
:::

:::retenir
- Un **traitement de texte** (LibreOffice Writer, Word) permet de saisir, mettre en forme et enregistrer des documents.
- **Mise en forme caractères** : gras, italique, souligné, police, taille, couleur.
- **Mise en forme paragraphes** : alignement (gauche, centré, droit, justifié), interligne, retraits.
- Les **styles** permettent une mise en forme cohérente et la génération d'une table des matières automatique.
- **PDF** = format de partage qui préserve la mise en page. **.odt/.docx** = format de travail modifiable.
- Raccourci essentiel : **Ctrl + S** pour enregistrer souvent !
:::
