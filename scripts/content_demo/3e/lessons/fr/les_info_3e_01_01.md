# Matériel informatique : composants et rôles

Un ordinateur n'est pas une boîte noire mystérieuse : c'est un ensemble de composants physiques qui travaillent en harmonie. Connaître ces composants, c'est comprendre pourquoi un ordinateur est rapide ou lent, et comment choisir le bon matériel pour ses besoins.

:::definition
L'**informatique** est la science du traitement automatique de l'information à l'aide de machines. Un **ordinateur** est une machine électronique programmable capable d'**acquérir**, de **stocker**, de **traiter** et de **restituer** des informations selon des instructions organisées en **programme**.
:::

## Les composants internes

Tous les composants internes d'un ordinateur sont reliés entre eux par la **carte mère** (*motherboard*), le circuit imprimé principal du système.

:::propriete
Tout ordinateur comprend quatre éléments fondamentaux :

- **Le processeur (CPU)** : il exécute les instructions des programmes — c'est le cerveau.
- **La mémoire vive (RAM)** : stockage temporaire des données en cours d'utilisation.
- **La mémoire de masse** (disque dur HDD ou SSD) : stockage permanent des fichiers et systèmes.
- **La carte mère** : relie et fait communiquer tous les composants.
:::

## Le processeur (CPU)

:::definition
Le **processeur** (*Central Processing Unit*, CPU) est le composant qui effectue tous les calculs. Il lit les instructions des programmes, les décode et les exécute. Sa performance se mesure en :

- **GHz (gigahertz)** : nombre d'opérations par seconde (1 GHz = 1 milliard d'opérations/s).
- **Nombre de cœurs** : un processeur à 4 cœurs peut traiter 4 tâches simultanément.

Exemple : un processeur Intel Core i5 à 3,6 GHz avec 6 cœurs.
:::

## La mémoire vive (RAM)

:::definition
La **mémoire vive** (RAM — *Random Access Memory*) est une mémoire **temporaire** et **volatile** : elle contient les programmes et données **en cours d'utilisation**. Plus la RAM est grande, plus l'ordinateur peut faire fonctionner de programmes en même temps sans ralentir.

**Son contenu s'efface entièrement à l'extinction de l'ordinateur.**
:::

:::exemple
Tu travailles sur un devoir dans LibreOffice Writer. Le fichier est chargé depuis le disque dur vers la RAM pour être modifié rapidement. Quand tu enregistres (Ctrl + S), les modifications retournent sur le disque. Si tu éteignes sans enregistrer, tout ce qui était dans la RAM est définitivement perdu.
:::

## La mémoire de masse

:::propriete
Les **unités de stockage** conservent les données de façon permanente, même sans alimentation :

| Type | Vitesse | Caractéristiques |
|---|---|---|
| Disque dur (HDD) | Lent | Capacité élevée, moins cher, fragile aux chocs |
| Disque SSD | Rapide | Résistant, silencieux, plus cher |
| Clé USB | Variable | Portable, faible capacité |
| DVD/Blu-Ray | Lent | Lecture seule ou inscriptible, non modifiable |
:::

## Les périphériques

:::definition
Les **périphériques** sont des appareils externes connectés à l'ordinateur. On les classe en trois catégories :

| Catégorie | Rôle | Exemples |
|---|---|---|
| **Entrée** | Envoient des données vers l'ordinateur | Clavier, souris, scanner, microphone, webcam |
| **Sortie** | Reçoivent des données de l'ordinateur | Écran, imprimante, haut-parleurs, vidéoprojecteur |
| **Entrée/Sortie** | Les deux sens | Clé USB, disque dur externe, carte réseau, écran tactile |
:::

## Unités de mesure de l'information

:::propriete
L'information numérique se mesure en **bits** et en **octets** :

| Unité | Symbole | Valeur |
|---|---|---|
| Bit | b | Unité de base : vaut 0 ou 1 |
| Octet | o (B en anglais) | 8 bits |
| Kilooctet | Ko | 1 024 octets |
| Mégaoctet | Mo | 1 024 Ko ≈ 1 million d'octets |
| Gigaoctet | Go | 1 024 Mo ≈ 1 milliard d'octets |
| Téraoctet | To | 1 024 Go |
:::

:::attention
Ne pas confondre **bit** (b minuscule) et **octet** (o minuscule ou **B** majuscule en anglais). Un débit internet de **8 Mbit/s** ne donne que **1 Mo/s** de téléchargement effectif, car 8 bits = 1 octet.
:::

:::methode
Pour évaluer un ordinateur avant achat :

1. **Définir l'usage** : bureautique légère, création graphique, jeux vidéo, développement ?
2. **Vérifier le processeur** : fréquence (GHz) + nombre de cœurs selon l'usage.
3. **Vérifier la RAM** : 4 Go minimum (bureautique), 8 Go recommandé, 16 Go pour la création.
4. **Vérifier le stockage** : SSD de préférence (rapidité), HDD pour grande capacité à bas coût.
5. **Vérifier la connectivité** : ports USB 3.0, Wi-Fi, HDMI, Bluetooth.
:::

:::retenir
- Un ordinateur = **processeur** (calcul) + **RAM** (temporaire) + **stockage** (permanent) + **carte mère** (liaison) + **périphériques** (communication).
- La **RAM est volatile** : tout s'efface à l'extinction. Le **disque dur/SSD** est permanent.
- **1 octet = 8 bits** ; 1 Ko = 1 024 o ; 1 Mo = 1 024 Ko ; 1 Go = 1 024 Mo.
- Périphériques d'**entrée** (clavier, souris), de **sortie** (écran, imprimante), d'**entrée-sortie** (clé USB).
:::
