# Systèmes d'exploitation et logiciels

Sans logiciel, le matériel informatique est inerte. C'est le système d'exploitation qui donne vie à l'ordinateur en gérant toutes ses ressources et en permettant aux autres logiciels de fonctionner.

:::definition
Un **logiciel** est un programme ou un ensemble de programmes permettant à l'ordinateur d'effectuer des tâches spécifiques. On distingue deux grandes familles :

- **Logiciels système** : gèrent le matériel et servent de base aux autres logiciels.
- **Logiciels d'application** : permettent à l'utilisateur d'accomplir des tâches précises (rédiger, naviguer, jouer, créer…).
:::

## Le système d'exploitation (SE)

:::definition
Le **système d'exploitation** (SE) est le logiciel de base indispensable au fonctionnement d'un ordinateur. Il assure trois rôles principaux :

1. **Gestion des ressources matérielles** : processeur, mémoire, disque dur, périphériques.
2. **Interface entre l'utilisateur et la machine** : bureau, fenêtres, icônes, menus.
3. **Gestion des fichiers** : création, organisation, lecture et suppression de fichiers et dossiers.

Sans système d'exploitation, aucun autre logiciel ne peut fonctionner.
:::

## Exemples de systèmes d'exploitation

:::propriete
| Système | Éditeur | Caractéristiques |
|---|---|---|
| **Windows 11** | Microsoft | Payant, très répandu sur PC bureautique |
| **macOS** | Apple | Payant, exclusif aux ordinateurs Mac |
| **Ubuntu Linux** | Canonical | **Gratuit et libre** (open source), stable, sécurisé |
| **Android** | Google | Open source, dominant sur smartphones/tablettes |
| **iOS** | Apple | Propriétaire, exclusif aux iPhone/iPad |

Un logiciel **libre** (*open source*) est un logiciel dont le code source est accessible à tous : on peut le lire, le modifier et le redistribuer librement.
:::

## L'arborescence de fichiers

:::definition
Les fichiers sont organisés en **arborescence** : une structure hiérarchique de **dossiers** (répertoires) contenant des fichiers et d'autres dossiers.

- **Dossier racine** : le point de départ de l'arborescence (C:\ sous Windows, / sous Linux).
- **Dossier** : conteneur virtuel regroupant des fichiers et sous-dossiers.
- **Fichier** : document ou programme stocké sur le disque (texte, image, vidéo, programme…).

Exemple d'arborescence :
```
C:\
├── Utilisateurs\
│   └── Amara\
│       ├── Documents\
│       │   └── Devoir_Maths.docx
│       └── Images\
│           └── Photo_classe.jpg
└── Programme Files\
    └── LibreOffice\
```
:::

:::propriete
| Catégorie | Exemples | Usage |
|---|---|---|
| Traitement de texte | LibreOffice Writer, Word | Rédiger des documents |

:::

## Logiciel libre vs logiciel propriétaire

:::propriete
| Critère | Logiciel libre | Logiciel propriétaire |
|---|---|---|
| Code source | Accessible à tous | Secret |
| Coût | Généralement gratuit | Payant (licence) |
| Modification | Autorisée | Interdite |
| Exemples | LibreOffice, Firefox, VLC, Linux | Windows, Microsoft Office, Adobe Photoshop |
:::

:::attention
**Logiciel gratuit ≠ logiciel libre.** Un logiciel peut être gratuit (freeware) sans être libre : le code source reste secret et on ne peut pas le modifier. À l'inverse, un logiciel libre n'est pas forcément gratuit, même si c'est souvent le cas.
:::

:::methode
Pour trouver la bonne alternative libre à un logiciel propriétaire :

1. Identifier la tâche à accomplir (traitement de texte, montage vidéo, retouche photo…).
2. Rechercher «alternative libre à [nom du logiciel]».
3. Vérifier la compatibilité avec les formats de fichiers utilisés (ex : .docx pour Word).
4. Tester le logiciel (la plupart sont téléchargeables gratuitement sur le site officiel).
:::

:::retenir
- Un **logiciel** est un programme permettant d'accomplir une tâche sur ordinateur.
- Le **système d'exploitation** gère les ressources matérielles et sert de base à tous les autres logiciels. Exemples : Windows, Linux, macOS, Android.
- L'**arborescence de fichiers** organise les données en dossiers et sous-dossiers hiérarchisés.
- **Logiciel libre** = code source accessible, modifiable, redistribuable. Exemple : Ubuntu Linux, LibreOffice.
:::
