# Salutations et présentations

Savoir se saluer et se présenter est la première étape pour communiquer en mandarin. En Chine, les salutations varient selon l'heure de la journée et le degré de formalité de la relation.

## Les salutations

:::propriete
| Mandarin | Pinyin | Signification | Contexte |
|---|---|---|---|
| 你好 | nǐ hǎo | bonjour | salutation universelle, informelle |
| 您好 | nín hǎo | bonjour (formel) | à un adulte, un supérieur, signe de respect |
| 早上好 | zǎoshang hǎo | bonjour (matin) | jusqu'à environ 10h |
| 下午好 | xiàwǔ hǎo | bonjour (après-midi) | de 12h à 18h |
| 晚上好 | wǎnshang hǎo | bonsoir | après 18h |
| 你好吗？| nǐ hǎo ma? | tu vas bien ? | demander des nouvelles |
| 很好，谢谢 | hěn hǎo, xièxiè | très bien, merci | réponse standard |
| 再见 | zàijiàn | au revoir | formule d'adieu |
| 明天见 | míngtiān jiàn | à demain | «on se voit demain» |
| 拜拜 | bāibāi | bye-bye | informel, entre amis (anglicisme) |
:::

## Se présenter

:::methode
**Structure de base pour se présenter :**

1. **Nom** : 我叫... wǒ jiào... = je m'appelle...
   - 我叫 Amara。Wǒ jiào Amara. = Je m'appelle Amara.

2. **Nationalité** : 我是...人 wǒ shì... rén = je suis... (nationalité)
   - 我是喀麦隆人。Wǒ shì Kāmàilóng rén. = Je suis Camerounais(e).
   - 我来自喀麦隆。Wǒ láizì Kāmàilóng. = Je viens du Cameroun.

3. **Âge** : 我...岁 wǒ... suì = j'ai... ans
   - 我十五岁。Wǒ shíwǔ suì. = J'ai 15 ans. (岁 suì = classifier pour les années d'âge)

4. **École** : 我是...学生 wǒ shì... xuéshēng = je suis élève à...
   - 我是初三学生。Wǒ shì chūsān xuéshēng. = Je suis en 3ème.
:::

## Poser des questions sur quelqu'un

:::propriete
| Question | Pinyin | Signification |
|---|---|---|
| 你叫什么名字？| nǐ jiào shénme míngzi? | comment tu t'appelles ? |
| 你是哪国人？| nǐ shì nǎ guó rén? | tu es de quel pays ? |
| 你从哪里来？| nǐ cóng nǎlǐ lái? | d'où viens-tu ? |
| 你多大了？| nǐ duō dà le? | quel âge as-tu ? |
| 你在哪个学校？| nǐ zài nǎge xuéxiào? | tu es dans quelle école ? |
:::

## Les formules de politesse

:::propriete
| Mandarin | Pinyin | Signification |
|---|---|---|
| 请 | qǐng | s'il vous plaît / veuillez |
| 谢谢 | xièxiè | merci |
| 非常感谢 | fēicháng gǎnxiè | merci beaucoup (formel) |
| 不客气 | bù kèqì | de rien (litt. «pas de cérémonie») |
| 没关系 | méi guānxi | ce n'est pas grave / il n'y a pas de souci |
| 对不起 | duìbuqǐ | pardon / désolé(e) |
| 请问 | qǐngwèn | excusez-moi / puis-je vous demander... |

:::exemple
**Dialogue : Amara rencontre Li Ming au lycée**

A : 你好！我叫 Amara。你叫什么名字？
    Nǐ hǎo! Wǒ jiào Amara. Nǐ jiào shénme míngzi?
    Bonjour ! Je m'appelle Amara. Comment tu t'appelles ?

L : 你好，Amara！我叫李明。很高兴认识你！
    Nǐ hǎo, Amara! Wǒ jiào Lǐ Míng. Hěn gāoxìng rènshi nǐ!
    Bonjour, Amara ! Je m'appelle Li Ming. Enchanté(e) !

A : 认识你也很高兴！你是哪国人？
    Rènshi nǐ yě hěn gāoxìng! Nǐ shì nǎ guó rén?
    Enchanté(e) aussi ! Tu es de quel pays ?

L : 我是中国人。你呢？
    Wǒ shì Zhōngguó rén. Nǐ ne?
    Je suis Chinois. Et toi ?

A : 我是喀麦隆人。我来自雅温得。
    Wǒ shì Kāmàilóng rén. Wǒ láizì Yǎwēndé.
    Je suis Camerounaise. Je viens de Yaoundé.

L : 太好了！再见！
    Tài hǎo le! Zàijiàn!
    Super ! Au revoir !
:::

:::attention
- 很高兴认识你 (hěn gāoxìng rènshi nǐ) = enchanté(e) de te connaître — à dire à la fin d'une présentation, pas au début.
- 您 nín est la forme de politesse de 你 nǐ — à utiliser avec les adultes, les professeurs et les personnes âgées.
:::

:::retenir
- Salutations selon l'heure : 早上好 (matin), 下午好 (après-midi), 晚上好 (soir).
- 你好 = informel ; 您好 = formel (adultes, supérieurs).
- Se présenter : 我叫... (nom) + 我是...人 (nationalité) + 我...岁 (âge).
- Politesse : 谢谢 (merci), 不客气 (de rien), 对不起 (pardon), 没关系 (pas grave).
- 很高兴认识你 = enchanté(e) de faire ta connaissance.
:::
