# Le temps et les dates (时间 shíjiān)

Savoir donner l'heure, les jours et les dates est indispensable pour la vie quotidienne. En mandarin, la structure temporelle suit une logique du plus grand au plus petit : année → mois → jour → heure → minute.

## Demander et donner l'heure

:::methode
**Structure pour exprimer l'heure :**

| Expression | Pinyin | Sens |
|---|---|---|
| 现在几点？| xiànzài jǐ diǎn? | Quelle heure est-il ? (maintenant) |
| 几点了？| jǐ diǎn le? | Quelle heure est-il ? (plus informel) |
| ...点 | ...diǎn | ...heure(s) |
| ...点半 | ...diǎn bàn | ...h30 (半 bàn = moitié) |
| ...点一刻 | ...diǎn yī kè | ...h15 (一刻 yī kè = un quart) |
| ...点三刻 | ...diǎn sān kè | ...h45 (trois quarts) |
| 差...分...点 | chā...fēn...diǎn | ...h moins...min (差 chā = manque) |
| 上午 | shàngwǔ | matin (AM) |
| 下午 | xiàwǔ | après-midi (PM) |
| 晚上 | wǎnshang | soir |
:::

:::propriete
**Exemples d'heures :**

| Heure | Mandarin | Pinyin |
|---|---|---|
| 3h00 | 三点 | sān diǎn |
| 3h30 | 三点半 | sān diǎn bàn |
| 3h15 | 三点一刻 | sān diǎn yī kè |
| 3h45 | 三点三刻 / 差一刻四点 | sān diǎn sān kè |
| 8h55 | 差五分九点 | chā wǔ fēn jiǔ diǎn |
| 12h00 (midi) | 中午十二点 | zhōngwǔ shí'èr diǎn |
| 0h00 (minuit) | 零点 / 午夜 | líng diǎn / wǔyè |
:::

## Les jours de la semaine

:::propriete
Le mandarin utilise 星期 xīngqī (ou 周 zhōu) suivi d'un chiffre :

| Jour | Caractère | Pinyin | Abréviation |
|---|---|---|---|
| Lundi | 星期一 | xīngqīyī | 周一 zhōuyī |
| Mardi | 星期二 | xīngqī'èr | 周二 |
| Mercredi | 星期三 | xīngqīsān | 周三 |
| Jeudi | 星期四 | xīngqīsì | 周四 |
| Vendredi | 星期五 | xīngqīwǔ | 周五 |
| Samedi | 星期六 | xīngqīliù | 周六 |
| Dimanche | 星期日 / 星期天 | xīngqīrì / xīngqītiān | 周日 |

Demander le jour : 今天星期几？Jīntiān xīngqī jǐ? = On est quel jour ?
:::

## Les mois et les dates

:::propriete
**Les mois** = chiffre + 月 yuè :

| Mois | Caractère | Pinyin |
|---|---|---|
| Janvier | 一月 | yīyuè |
| Février | 二月 | èryuè |
| Mars | 三月 | sānyuè |
| Avril | 四月 | sìyuè |
| Mai | 五月 | wǔyuè |
| Juin | 六月 | liùyuè |
| Juillet | 七月 | qīyuè |
| Août | 八月 | bāyuè |
| Septembre | 九月 | jiǔyuè |
| Octobre | 十月 | shíyuè |
| Novembre | 十一月 | shíyīyuè |
| Décembre | 十二月 | shí'èryuè |

**Les dates** — structure : année + 年 + mois + 月 + jour + 日/号

Exemple : 二零二六年六月三十日 = 30 juin 2026
- 年 nián = an/année
- 月 yuè = mois
- 日 rì (écrit) / 号 hào (oral) = jour

Demander la date : 今天几月几号？Jīntiān jǐ yuè jǐ hào? = On est le combien ?
:::

## Mots relatifs au temps

:::propriete
| Mot | Caractère | Pinyin |
|---|---|---|
| Aujourd'hui | 今天 | jīntiān |
| Demain | 明天 | míngtiān |
| Après-demain | 后天 | hòutiān |
| Hier | 昨天 | zuótiān |
| Avant-hier | 前天 | qiántiān |
| Cette semaine | 这个星期 | zhège xīngqī |
| La semaine prochaine | 下个星期 | xià gè xīngqī |
| La semaine dernière | 上个星期 | shàng gè xīngqī |
| Ce mois-ci | 这个月 | zhège yuè |
| Cette année | 今年 | jīnnián |
| L'année prochaine | 明年 | míngnián |
| L'année dernière | 去年 | qùnián |
:::

:::exemple
**Dialogue :**

Amara : 现在几点？
         Xiànzài jǐ diǎn?
         Quelle heure est-il ?

Kevin : 下午三点半。你今天有课吗？
         Xiàwǔ sān diǎn bàn. Nǐ jīntiān yǒu kè ma?
         15h30. Tu as cours aujourd'hui ?

Amara : 有，下午四点有数学课。今天是几月几号？
         Yǒu, xiàwǔ sì diǎn yǒu shùxué kè. Jīntiān shì jǐ yuè jǐ hào?
         Oui, à 16h j'ai un cours de maths. On est le combien ?

Kevin : 今天是六月三十号，星期二。
         Jīntiān shì liù yuè sānshí hào, xīngqī'èr.
         Aujourd'hui c'est le 30 juin, mardi.
:::

:::attention
En mandarin, l'ordre dans une date est **toujours du plus grand au plus petit** :
Année (年) → Mois (月) → Jour (日) → Heure (点) → Minute (分)

Exemple : 2026年6月30日下午3点30分 = le 30 juin 2026 à 15h30.

C'est l'inverse de l'ordre français (jour/mois/année) mais la même logique que l'ordre informatique (YYYY-MM-DD).
:::

:::retenir
- Heure : ...点 (heure), 点半 (h30), 点一刻 (h15), 差...分 (moins...min).
- Jours : 星期 + chiffre (一 lundi → 六 samedi), 星期日 dimanche.
- Mois : chiffre + 月 (一月 janvier... 十二月 décembre).
- Dates : année + 年 + mois + 月 + jour + 号 (du grand au petit).
- Repères : 今天 (auj.), 明天 (dem.), 昨天 (hier), 后天 (après-dem.).
:::
