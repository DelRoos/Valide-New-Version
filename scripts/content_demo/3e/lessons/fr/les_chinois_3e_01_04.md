# Les chiffres et les mots-mesure

En mandarin, compter ne suffit pas : il faut aussi maîtriser les **mots-mesure** (ou classifieurs), ces petits mots obligatoires que l'on place entre un nombre et un nom. C'est l'une des particularités les plus importantes de la grammaire chinoise.

## Les nombres composés

:::methode
**De 11 à 99 :** dizaine + unité

| Nombre | Mandarin | Pinyin |
|---|---|---|
| 11 | 十一 | shíyī |
| 20 | 二十 | èrshí |
| 21 | 二十一 | èrshíyī |
| 50 | 五十 | wǔshí |
| 99 | 九十九 | jiǔshíjiǔ |

**Au-delà de 99 :**
- 100 = 百 bǎi → 一百 yī bǎi
- 1 000 = 千 qiān → 一千 yī qiān
- 10 000 = 万 wàn → 一万 yī wàn (unité unique au chinois !)

Exemples : 256 = 二百五十六 (èrbǎi wǔshíliù) ; 2 000 = 两千 (liǎng qiān)
:::

:::attention
**Règle importante : 两 liǎng vs 二 èr**

- Devant un classifieur, on utilise **两 liǎng** et non 二 èr.
- ✅ 两个人 liǎng gè rén = deux personnes
- ❌ 二个人 — incorrect
- 二 s'utilise seul (compter, numéros) : 二月 (février), 二楼 (2e étage), 第二 (deuxième).

Exception : dans les nombres composés, 二 reste : 二十 (20), 一百二 (102).
:::

## Les mots-mesure (classifieurs 量词)

:::definition
En mandarin, on **ne peut pas** placer un nombre directement devant un nom. Il faut intercaler un **mot-mesure** (量词 liàngcí), aussi appelé classifieur.

Structure obligatoire : **nombre + classifieur + nom**

Exemple : «deux livres» = 两 (2) + 本 (classifieur des livres) + 书 (livre) = **两本书**
:::

:::propriete
| Classifieur | Pinyin | Utilisé pour | Exemple |
|---|---|---|---|
| 个 | gè | Usage général (personnes, objets courants) | 一个苹果 yī gè píngguǒ (une pomme) |
| 本 | běn | Livres, cahiers, magazines | 两本书 liǎng běn shū (deux livres) |
| 张 | zhāng | Objets plats (papier, photo, table, billet) | 三张纸 sān zhāng zhǐ (trois feuilles) |
| 只 | zhī | Animaux, certains petits objets | 一只猫 yī zhī māo (un chat) |
| 条 | tiáo | Objets longs et flexibles (poisson, route, pantalon) | 一条鱼 yī tiáo yú (un poisson) |
| 杯 | bēi | Tasses, verres (boissons) | 一杯水 yī bēi shuǐ (un verre d'eau) |
| 碗 | wǎn | Bols (riz, soupe, nouilles) | 一碗面 yī wǎn miàn (un bol de nouilles) |
| 口 | kǒu | Membres d'une famille | 三口人 sān kǒu rén (3 personnes dans la famille) |
| 件 | jiàn | Vêtements (hauts), affaires | 两件衬衫 liǎng jiàn chènshān (deux chemises) |
| 位 | wèi | Personnes (poli/respectueux) | 一位老师 yī wèi lǎoshī (un professeur) |
:::

## La culture des chiffres en Chine

:::propriete
| Chiffre | Perception | Raison |
|---|---|---|
| 8 (八 bā) | ✅ Très porte-bonheur | Sonne comme 发 fā (prospérité, s'enrichir) |
| 6 (六 liù) | ✅ Porte-bonheur | Signifie la fluidité, le succès continu |
| 9 (九 jiǔ) | ✅ Porte-bonheur | Sonne comme 久 jiǔ (longtemps, éternité) |
| 4 (四 sì) | ❌ Porte-malheur | Sonne comme 死 sǐ (mort) — évité dans les étages, numéros de chambre |
| 88 | ✅ Très populaire | Double prospérité — souvent dans les SMS et les prix |

Les numéros de téléphone, plaques d'immatriculation et appartements contenant le chiffre 8 sont très prisés en Chine !
:::

:::exemple
Complète avec le bon classifieur (个/本/张/只/杯) :

1. 一＿书 → 一**本**书 (un livre)
2. 两＿学生 → 两**个**学生 (deux élèves)
3. 三＿照片 → 三**张**照片 (trois photos)
4. 一＿狗 → 一**只**狗 (un chien)
5. 两＿茶 → 两**杯**茶 (deux tasses de thé)
:::

:::retenir
- Chiffres 0-10 à mémoriser ; au-delà : dizaine + unité (十一, 二十...).
- Grandes unités : 百 (100), 千 (1 000), 万 (10 000).
- **两 liǎng** remplace 二 devant un classifieur.
- Structure obligatoire : **nombre + classifieur + nom**.
- Classifieurs clés : 个 (général), 本 (livres), 张 (plat), 只 (animaux), 杯 (boissons).
- 8 = porte-bonheur ; 4 = porte-malheur en culture chinoise.
:::
