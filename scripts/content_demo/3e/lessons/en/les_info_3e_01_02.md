# Operating systems and software

Without software, computer hardware is inert. It is the operating system that brings the computer to life by managing all its resources and allowing other software to function.

:::definition
**Software** is a program or set of programs that enables a computer to perform specific tasks. Two main families are distinguished:

- **System software**: manages hardware and serves as the foundation for other software.
- **Application software**: allows users to perform specific tasks (write, browse, play, create…).
:::

## The operating system (OS)

:::definition
The **operating system** (OS) is the essential basic software for a computer to function. It fulfils three main roles:

1. **Hardware resource management**: processor, memory, hard disk, peripherals.
2. **Interface between user and machine**: desktop, windows, icons, menus.
3. **File management**: creating, organising, reading and deleting files and folders.

Without an operating system, no other software can function.
:::

## Examples of operating systems

:::propriete
| System | Developer | Characteristics |
|---|---|---|
| **Windows 11** | Microsoft | Paid, very widespread on desktop PCs |
| **macOS** | Apple | Paid, exclusive to Mac computers |
| **Ubuntu Linux** | Canonical | **Free and open source**, stable, secure |
| **Android** | Google | Open source, dominant on smartphones/tablets |
| **iOS** | Apple | Proprietary, exclusive to iPhone/iPad |

**Open source** software is software whose source code is accessible to everyone: it can be read, modified and freely redistributed.
:::

## The file tree structure

:::definition
Files are organised in a **tree structure**: a hierarchical structure of **folders** (directories) containing files and other folders.

- **Root folder**: the starting point of the tree (C:\ on Windows, / on Linux).
- **Folder**: virtual container grouping files and subfolders.
- **File**: document or program stored on disk (text, image, video, program…).

Example tree structure:
```
C:\
├── Users\
│   └── Amara\
│       ├── Documents\
│       │   └── Maths_Homework.docx
│       └── Pictures\
│           └── Class_photo.jpg
└── Program Files\
    └── LibreOffice\
```
:::

## Free software vs proprietary software

:::propriete
| Criterion | Free/Open source software | Proprietary software |
|---|---|---|
| Source code | Accessible to all | Secret |
| Cost | Generally free | Paid (licence) |
| Modification | Permitted | Prohibited |
| Examples | LibreOffice, Firefox, VLC, Linux | Windows, Microsoft Office, Adobe Photoshop |
:::

:::attention
**Free of charge ≠ free/open source.** Software can be free of charge (freeware) without being open source: the source code remains secret and cannot be modified. Conversely, open source software is not necessarily free of charge, although it often is.
:::

:::methode
To find the right open source alternative to proprietary software:

1. Identify the task to be accomplished (word processing, video editing, photo retouching…).
2. Search for «open source alternative to [software name]».
3. Check compatibility with the file formats used (e.g. .docx for Word).
4. Test the software (most are freely downloadable from the official website).
:::

:::retenir
- **Software** is a program that enables tasks to be performed on a computer.
- The **operating system** manages hardware resources and serves as the foundation for all other software. Examples: Windows, Linux, macOS, Android.
- The **file tree structure** organises data into hierarchical folders and subfolders.
- **Open source software** = accessible, modifiable, redistributable source code. Examples: Ubuntu Linux, LibreOffice.
:::
