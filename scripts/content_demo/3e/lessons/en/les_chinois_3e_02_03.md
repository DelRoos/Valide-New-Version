# Time and dates (时间 shíjiān)

Knowing how to give the time, days and dates is essential for daily life. In Mandarin, the time structure follows a logic from largest to smallest: year → month → day → hour → minute.

## Asking and telling the time

:::methode
**Structure for expressing the time:**

| Expression | Pinyin | Meaning |
|---|---|---|
| 现在几点？| xiànzài jǐ diǎn? | What time is it? (now) |
| 几点了？| jǐ diǎn le? | What time is it? (more informal) |
| ...点 | ...diǎn | ...o'clock |
| ...点半 | ...diǎn bàn | ...h30 (半 bàn = half) |
| ...点一刻 | ...diǎn yī kè | ...h15 (一刻 yī kè = a quarter) |
| ...点三刻 | ...diǎn sān kè | ...h45 (three quarters) |
| 差...分...点 | chā...fēn...diǎn | ...h minus...min (差 chā = lacking) |
| 上午 | shàngwǔ | morning (AM) |
| 下午 | xiàwǔ | afternoon (PM) |
| 晚上 | wǎnshang | evening |
:::

:::propriete
**Examples of times:**

| Time | Mandarin | Pinyin |
|---|---|---|
| 3:00 | 三点 | sān diǎn |
| 3:30 | 三点半 | sān diǎn bàn |
| 3:15 | 三点一刻 | sān diǎn yī kè |
| 3:45 | 三点三刻 / 差一刻四点 | sān diǎn sān kè |
| 8:55 | 差五分九点 | chā wǔ fēn jiǔ diǎn |
| 12:00 (noon) | 中午十二点 | zhōngwǔ shí'èr diǎn |
| 00:00 (midnight) | 零点 / 午夜 | líng diǎn / wǔyè |
:::

## Days of the week

:::propriete
Mandarin uses 星期 xīngqī (or 周 zhōu) followed by a number:

| Day | Character | Pinyin | Abbreviation |
|---|---|---|---|
| Monday | 星期一 | xīngqīyī | 周一 zhōuyī |
| Tuesday | 星期二 | xīngqī'èr | 周二 |
| Wednesday | 星期三 | xīngqīsān | 周三 |
| Thursday | 星期四 | xīngqīsì | 周四 |
| Friday | 星期五 | xīngqīwǔ | 周五 |
| Saturday | 星期六 | xīngqīliù | 周六 |
| Sunday | 星期日 / 星期天 | xīngqīrì / xīngqītiān | 周日 |

Asking the day: 今天星期几？Jīntiān xīngqī jǐ? = What day is it today?
:::

## Months and dates

:::propriete
**Months** = number + 月 yuè:

| Month | Character | Pinyin |
|---|---|---|
| January | 一月 | yīyuè |
| February | 二月 | èryuè |
| March | 三月 | sānyuè |
| April | 四月 | sìyuè |
| May | 五月 | wǔyuè |
| June | 六月 | liùyuè |
| July | 七月 | qīyuè |
| August | 八月 | bāyuè |
| September | 九月 | jiǔyuè |
| October | 十月 | shíyuè |
| November | 十一月 | shíyīyuè |
| December | 十二月 | shí'èryuè |

**Dates** — structure: year + 年 + month + 月 + day + 日/号

Example: 二零二六年六月三十日 = 30 June 2026
- 年 nián = year
- 月 yuè = month
- 日 rì (written) / 号 hào (spoken) = day

Asking the date: 今天几月几号？Jīntiān jǐ yuè jǐ hào? = What is today's date?
:::

## Time reference words

:::propriete
| Word | Character | Pinyin |
|---|---|---|
| Today | 今天 | jīntiān |
| Tomorrow | 明天 | míngtiān |
| Day after tomorrow | 后天 | hòutiān |
| Yesterday | 昨天 | zuótiān |
| Day before yesterday | 前天 | qiántiān |
| This week | 这个星期 | zhège xīngqī |
| Next week | 下个星期 | xià gè xīngqī |
| Last week | 上个星期 | shàng gè xīngqī |
| This month | 这个月 | zhège yuè |
| This year | 今年 | jīnnián |
| Next year | 明年 | míngnián |
| Last year | 去年 | qùnián |
:::

:::exemple
**Dialogue:**

Amara: 现在几点？
        Xiànzài jǐ diǎn?
        What time is it?

Kevin: 下午三点半。你今天有课吗？
        Xiàwǔ sān diǎn bàn. Nǐ jīntiān yǒu kè ma?
        3:30 pm. Do you have class today?

Amara: 有，下午四点有数学课。今天是几月几号？
        Yǒu, xiàwǔ sì diǎn yǒu shùxué kè. Jīntiān shì jǐ yuè jǐ hào?
        Yes, I have maths at 4pm. What is today's date?

Kevin: 今天是六月三十号，星期二。
        Jīntiān shì liù yuè sānshí hào, xīngqī'èr.
        Today is the 30th of June, Tuesday.
:::

:::attention
In Mandarin, the order in a date is **always from largest to smallest**:
Year (年) → Month (月) → Day (日) → Hour (点) → Minute (分)

Example: 2026年6月30日下午3点30分 = 30 June 2026 at 3:30pm.

This is the opposite of the British/French order (day/month/year) but follows the same logic as the ISO date format (YYYY-MM-DD).
:::

:::retenir
- Time: ...点 (o'clock), 点半 (h30), 点一刻 (h15), 差...分 (minus...min).
- Days: 星期 + number (一 Monday → 六 Saturday), 星期日 Sunday.
- Months: number + 月 (一月 January... 十二月 December).
- Dates: year + 年 + month + 月 + day + 号 (largest to smallest).
- Reference words: 今天 (today), 明天 (tomorrow), 昨天 (yesterday), 后天 (day after tomorrow).
:::
