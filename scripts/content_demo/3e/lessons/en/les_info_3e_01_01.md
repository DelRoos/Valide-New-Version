# Computer hardware: components and roles

A computer is not a mysterious black box: it is a set of physical components working in harmony. Understanding these components means understanding why a computer is fast or slow, and how to choose the right hardware for your needs.

:::definition
**Computing** is the science of automatic information processing using machines. A **computer** is a programmable electronic machine capable of **acquiring**, **storing**, **processing** and **outputting** information according to instructions organised into a **program**.
:::

## Internal components

All internal components of a computer are connected to each other by the **motherboard**, the system's main circuit board.

:::propriete
Every computer contains four fundamental elements:

- **The processor (CPU)**: it executes program instructions — it is the brain.
- **Random Access Memory (RAM)**: temporary storage of data currently in use.
- **Mass storage** (hard disk HDD or SSD): permanent storage of files and systems.
- **The motherboard**: connects and enables communication between all components.
:::

## The processor (CPU)

:::definition
The **processor** (*Central Processing Unit*, CPU) is the component that performs all calculations. It reads program instructions, decodes them and executes them. Its performance is measured by:

- **GHz (gigahertz)**: number of operations per second (1 GHz = 1 billion operations/s).
- **Number of cores**: a 4-core processor can handle 4 tasks simultaneously.

Example: an Intel Core i5 processor at 3.6 GHz with 6 cores.
:::

## Random Access Memory (RAM)

:::definition
**Random Access Memory** (RAM) is **temporary** and **volatile** memory: it contains the programs and data **currently in use**. The larger the RAM, the more programs the computer can run simultaneously without slowing down.

**Its contents are completely erased when the computer is switched off.**
:::

:::exemple
You are working on an assignment in LibreOffice Writer. The file is loaded from the hard drive into RAM so it can be edited quickly. When you save (Ctrl + S), the changes go back to the disk. If you switch off without saving, everything that was in RAM is permanently lost.
:::

## Mass storage

:::propriete
**Storage units** retain data permanently, even without power:

| Type | Speed | Characteristics |
|---|---|---|
| Hard disk (HDD) | Slow | High capacity, cheaper, sensitive to shocks |
| SSD disk | Fast | Resistant, silent, more expensive |
| USB key | Variable | Portable, low capacity |
| DVD/Blu-Ray | Slow | Read-only or writable, not rewritable |
:::

## Peripheral devices

:::definition
**Peripheral devices** are external devices connected to the computer. They fall into three categories:

| Category | Role | Examples |
|---|---|---|
| **Input** | Send data to the computer | Keyboard, mouse, scanner, microphone, webcam |
| **Output** | Receive data from the computer | Screen, printer, speakers, projector |
| **Input/Output** | Both directions | USB key, external hard drive, network card, touchscreen |
:::

## Units of measurement for information

:::propriete
Digital information is measured in **bits** and **bytes**:

| Unit | Symbol | Value |
|---|---|---|
| Bit | b | Basic unit: 0 or 1 |
| Byte | B | 8 bits |
| Kilobyte | KB | 1,024 bytes |
| Megabyte | MB | 1,024 KB ≈ 1 million bytes |
| Gigabyte | GB | 1,024 MB ≈ 1 billion bytes |
| Terabyte | TB | 1,024 GB |
:::

:::attention
Do not confuse **bit** (lowercase b) and **byte** (uppercase B). An Internet speed of **8 Mbit/s** only gives **1 MB/s** of actual download speed, because 8 bits = 1 byte.
:::

:::methode
To evaluate a computer before buying:

1. **Define the use**: light office work, graphic design, video games, programming?
2. **Check the processor**: frequency (GHz) + number of cores for the intended use.
3. **Check the RAM**: 4 GB minimum (office work), 8 GB recommended, 16 GB for creative work.
4. **Check storage**: SSD preferred (speed), HDD for large capacity at lower cost.
5. **Check connectivity**: USB 3.0 ports, Wi-Fi, HDMI, Bluetooth.
:::

:::retenir
- A computer = **processor** (calculation) + **RAM** (temporary) + **storage** (permanent) + **motherboard** (connection) + **peripherals** (communication).
- **RAM is volatile**: everything is erased when switched off. The **hard drive/SSD** is permanent.
- **1 byte = 8 bits**; 1 KB = 1,024 B; 1 MB = 1,024 KB; 1 GB = 1,024 MB.
- **Input** peripherals (keyboard, mouse), **output** (screen, printer), **input/output** (USB key).
:::
