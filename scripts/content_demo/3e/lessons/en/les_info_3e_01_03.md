# Word processing

Word processing is one of the most common uses of a computer. Mastering software such as LibreOffice Writer or Microsoft Word is an essential skill for school and professional life.

:::definition
A **word processing program** is software that allows you to enter, edit, format, save and print text documents. The most common are **LibreOffice Writer** (free and open source), **Microsoft Word** (paid) and **Google Docs** (online, free).
:::

## The word processor interface

:::propriete
A word processor interface includes:

- **The title bar**: displays the document and software name.
- **The ribbon / toolbar**: groups formatting functions (font, size, styles…).
- **The editing area**: the white space where text is typed.
- **The ruler**: allows adjustment of margins and tab stops.
- **The status bar**: displays page number, word count, language.
:::

## Character formatting

:::propriete
Character formatting changes the appearance of selected text:

| Formatting | Shortcut | Effect |
|---|---|---|
| **Bold** | Ctrl + B | **Bold text** |
| *Italic* | Ctrl + I | *Italic text* |
| Underline | Ctrl + U | Underlined text |
| Font | — | Change letter shape (Arial, Times…) |
| Size | — | Change size in points (12 pt standard) |
| Colour | — | Change text colour |
:::

## Paragraph formatting

:::propriete
Paragraph formatting acts on blocks of text:

| Option | Description |
|---|---|
| Left alignment | Text aligned to the left margin (default) |
| Centred alignment | Text centred on the page |
| Right alignment | Text aligned to the right margin |
| Justified | Text aligned on both sides (left AND right) |
| Line spacing | Space between lines (1, 1.5 or 2) |
| Indent | Offset of the paragraph from the margin |
:::

## Styles

:::definition
A **style** is a set of formatting settings (font, size, colour, alignment, line spacing…) saved under a name and applicable with a single click. The most common styles are:

- **Normal**: standard body text.
- **Heading 1**: main document title (large, bold).
- **Heading 2**: subtitle (slightly smaller).
- **Heading 3**: sub-subtitle.

Consistent use of styles allows automatic generation of a **table of contents**.
:::

:::exemple
To structure a presentation on the environment:
- Select «Introduction» → apply the **Heading 1** style.
- Select «1. Deforestation» → apply the **Heading 2** style.
- Select the body text → **Normal** style.
- Insert → Table of Contents → the software automatically generates the table with page numbers.
:::

## Saving and file formats

:::propriete
| Format | Extension | Use |
|---|---|---|
| Word document | .docx | Microsoft Office format, very widespread |
| ODF document | .odt | Open format (LibreOffice, OpenOffice) |
| PDF | .pdf | Final sharing, layout preserved |
| Plain text | .txt | Text without formatting |

> **Tip:** Work in .odt or .docx to edit the document. Export to PDF to share or print it.
:::

## Essential keyboard shortcuts

:::methode
Mastering these shortcuts allows much faster working:

| Action | Windows shortcut |
|---|---|
| Save | Ctrl + S |
| Undo | Ctrl + Z |
| Redo | Ctrl + Y |
| Copy | Ctrl + C |
| Cut | Ctrl + X |
| Paste | Ctrl + V |
| Select all | Ctrl + A |
| Print | Ctrl + P |
| Bold | Ctrl + B |
| Italic | Ctrl + I |
| Underline | Ctrl + U |
| Find | Ctrl + F |
:::

:::attention
Remember to save your work frequently with **Ctrl + S**. In the event of a power cut (very common in the Cameroonian context), any unsaved work is lost. LibreOffice has an automatic recovery feature, but it is not infallible.
:::

:::retenir
- A **word processor** (LibreOffice Writer, Word) allows you to type, format and save documents.
- **Character formatting**: bold, italic, underline, font, size, colour.
- **Paragraph formatting**: alignment (left, centred, right, justified), line spacing, indents.
- **Styles** enable consistent formatting and automatic generation of a table of contents.
- **PDF** = sharing format that preserves layout. **.odt/.docx** = editable working format.
- Essential shortcut: **Ctrl + S** to save frequently!
:::
