# Numbers and measure words

In Mandarin, counting is not enough: you also need to master **measure words** (or classifiers), the obligatory little words placed between a number and a noun. This is one of the most important features of Chinese grammar.

## Numbers from 0 to 10

:::propriete
| Number | Character | Pinyin | Note |
|---|---|---|---|
| 0 | 零 | líng | |
| 1 | 一 | yī | |
| 2 | 二 | èr | but use 两 liǎng before a classifier |
| 3 | 三 | sān | |
| 4 | 四 | sì | unlucky number (sounds like 死 sǐ = death) |
| 5 | 五 | wǔ | |
| 6 | 六 | liù | lucky number (= smooth, flowing) |
| 7 | 七 | qī | |
| 8 | 八 | bā | very lucky (sounds like 发 fā = prosperity) |
| 9 | 九 | jiǔ | lucky (sounds like 久 jiǔ = long time) |
| 10 | 十 | shí | |
:::

## Compound numbers

:::methode
**From 11 to 99:** tens + units

| Number | Mandarin | Pinyin |
|---|---|---|
| 11 | 十一 | shíyī |
| 20 | 二十 | èrshí |
| 21 | 二十一 | èrshíyī |
| 50 | 五十 | wǔshí |
| 99 | 九十九 | jiǔshíjiǔ |

**Beyond 99:**
- 100 = 百 bǎi → 一百 yī bǎi
- 1,000 = 千 qiān → 一千 yī qiān
- 10,000 = 万 wàn → 一万 yī wàn (a unique unit in Chinese!)

Examples: 256 = 二百五十六 (èrbǎi wǔshíliù); 2,000 = 两千 (liǎng qiān)
:::

:::attention
**Important rule: 两 liǎng vs 二 èr**

- Before a classifier, use **两 liǎng**, not 二 èr.
- ✅ 两个人 liǎng gè rén = two people
- ❌ 二个人 — incorrect
- 二 is used alone (counting, numbers): 二月 (February), 二楼 (2nd floor), 第二 (second).

Exception: in compound numbers, 二 remains: 二十 (20), 一百二 (102).
:::

## Measure words / classifiers (量词)

:::definition
In Mandarin, you **cannot** place a number directly before a noun. You must insert a **measure word** (量词 liàngcí), also called a classifier.

Obligatory structure: **number + classifier + noun**

Example: 'two books' = 两 (2) + 本 (classifier for books) + 书 (book) = **两本书**
:::

:::propriete
| Classifier | Pinyin | Used for | Example |
|---|---|---|---|
| 个 | gè | General (people, common objects) | 一个苹果 yī gè píngguǒ (one apple) |
| 本 | běn | Books, notebooks, magazines | 两本书 liǎng běn shū (two books) |
| 张 | zhāng | Flat objects (paper, photo, table, ticket) | 三张纸 sān zhāng zhǐ (three sheets of paper) |
| 只 | zhī | Animals, some small objects | 一只猫 yī zhī māo (one cat) |
| 条 | tiáo | Long, flexible things (fish, road, trousers) | 一条鱼 yī tiáo yú (one fish) |
| 杯 | bēi | Cups, glasses (drinks) | 一杯水 yī bēi shuǐ (a glass of water) |
| 碗 | wǎn | Bowls (rice, soup, noodles) | 一碗面 yī wǎn miàn (one bowl of noodles) |
| 口 | kǒu | Family members | 三口人 sān kǒu rén (3 people in the family) |
| 件 | jiàn | Items of clothing (tops), belongings | 两件衬衫 liǎng jiàn chènshān (two shirts) |
| 位 | wèi | People (polite/respectful) | 一位老师 yī wèi lǎoshī (a teacher) |
:::

## Number culture in China

:::propriete
| Number | Perception | Reason |
|---|---|---|
| 8 (八 bā) | ✅ Very lucky | Sounds like 发 fā (prosperity, to get rich) |
| 6 (六 liù) | ✅ Lucky | Means smooth progress, continuous success |
| 9 (九 jiǔ) | ✅ Lucky | Sounds like 久 jiǔ (long time, eternity) |
| 4 (四 sì) | ❌ Unlucky | Sounds like 死 sǐ (death) — avoided in floor numbers and room numbers |
| 88 | ✅ Very popular | Double prosperity — common in messages and prices |

Phone numbers, car registration plates and flats containing the number 8 are highly sought after in China!
:::

:::exemple
Fill in with the correct classifier (个/本/张/只/杯):

1. 一＿书 → 一**本**书 (one book)
2. 两＿学生 → 两**个**学生 (two students)
3. 三＿照片 → 三**张**照片 (three photos)
4. 一＿狗 → 一**只**狗 (one dog)
5. 两＿茶 → 两**杯**茶 (two cups of tea)
:::

:::retenir
- Numbers 0–10 to memorise; beyond: tens + units (十一, 二十...).
- Large units: 百 (100), 千 (1,000), 万 (10,000).
- **两 liǎng** replaces 二 before a classifier.
- Obligatory structure: **number + classifier + noun**.
- Key classifiers: 个 (general), 本 (books), 张 (flat), 只 (animals), 杯 (drinks).
- 8 = lucky; 4 = unlucky in Chinese culture.
:::
